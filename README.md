Interface graficzny do:

https://github.com/hamhobbypl/wavegenerator

Użycie
Kliknij Przeglądaj i wskaż plik JSON z definicją ćwiczenia (format opisany poniżej).
Ustaw parametry:
WPM – prędkość znakowa (słów na minutę, np. 20).
FWPM – prędkość Farnswortha (np. 12) – wydłuża przerwy między znakami.
Częstotliwość – ton CW w Hz (zalecane 400–800 Hz).
Sample rate – częstotliwość próbkowania (zwykle 44100).
Amplituda – głośność (0.0 – 1.0, domyślnie 0.35).
Cisza końcowa – dodatkowa cisza na końcu pliku (s).
Przerwa X, Y, Z – patrz niżej.
Wybierz miejsce zapisu pliku WAV.

Kliknij Generuj plik WAV.
